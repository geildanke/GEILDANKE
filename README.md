# GEILDANKE
Jekyll version of geildanke.com

## Installation

`bundle install`

## Development

`jekyll serve --watch``

## Build

`jekyll build`

## Deployment

Create a `_glynn.yml` from the template `_glynn.yml_template` and update the data according to your host.
Glynn will build the site and uplkoad it to the given directory.

`glynn`
